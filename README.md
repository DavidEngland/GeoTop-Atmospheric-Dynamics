# GeoTop-Atmospheric-Dynamics

Course-design repository for UAH material on geometric and topological methods in atmospheric and fluid dynamics.

This repo is organized around two working areas:

- `docs/` for syllabus material, lecture notes, project briefs, reference summaries, and course design drafts.
- `hw/` for homework sets, solution sketches, and grading support.

## Current Structure

### `docs/`
Use this folder for course-facing narrative and reference material:

- Course overview and syllabus
- Lecture notes and cheatsheets
- Project ideas and capstone planning
- Reading notes and source summaries
- TeX lecture/reference documents

### `hw/`
Use this folder for assignments and instructor support:

- Homework problem sets
- Draft solution keys
- Problem rewrites and grading notes

## Working Convention

When adding new material:

1. Put narrative/course-level material in `docs/`.
2. Put exercises, solutions, and assessment material in `hw/`.
3. Prefer small index files inside each folder over long flat lists of drafts.
4. Keep filenames descriptive and stable; use version suffixes only when revision history matters.
